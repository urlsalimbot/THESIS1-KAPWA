    ```
    export default {
      darkMode: "class",
      content: ['./index.html', './src/**/*.{js,ts,jsx,tsx}'],
      theme: { ... },
      plugins: [...]
    }
    ```
    Per D-12: Uses `class`-based dark mode so Phase 8 toggle can add/remove `.dark` on `<html>`.

    2. **Update fontFamily to CSS var references** (D-14): Change the existing `fontFamily` to:
    ```
    fontFamily: {
      heading: ['var(--font-heading)', 'Lexend', 'sans-serif'],
      body: ['var(--font-body)', 'Source Sans 3', '-apple-system', 'BlinkMacSystemFont', 'Segoe UI', 'sans-serif'],
    },
    ```
    The CSS var reference with fallback ensures the tokens defined in :root can be overridden (e.g. by a font-switching feature), while still falling back to the existing fonts.

    3. **Add fontSize extension** (D-13):
    ```
    fontSize: {
      xs: ['var(--font-size-xs)', { lineHeight: 'var(--line-height-none)' }],

      base: ['var(--font-size-base)', { lineHeight: 'var(--line-height-normal)' }],
      lg: ['var(--font-size-lg)', { lineHeight: 'var(--line-height-normal)' }],
      xl: ['var(--font-size-xl)', { lineHeight: 'var(--line-height-relaxed)' }],
      '2xl': ['var(--font-size-2xl)', { lineHeight: 'var(--line-height-relaxed)' }],
      '3xl': ['var(--font-size-3xl)', { lineHeight: 'var(--line-height-normal)' }],
    },
    ```

    4. **Add fontWeight extension** (D-13):
    ```
    fontWeight: {
      normal: 'var(--font-weight-normal)',
      medium: 'var(--font-weight-medium)',
      semibold: 'var(--font-weight-semibold)',
      bold: 'var(--font-weight-bold)',
    },
    ```

    5. **Add spacing extension** (D-13):
    ```
    spacing: {
      4: 'var(--spacing-4)',

      12: 'var(--spacing-12)',
      16: 'var(--spacing-16)',
      24: 'var(--spacing-24)',
      32: 'var(--spacing-32)',
      48: 'var(--spacing-48)',
      64: 'var(--spacing-64)',
    },
    ```
    These override Tailwind's default spacing values for these keys. Other spacing values (0, 1, 2, 3, 5, 6, 10, 20, 40, 56, 72, 80, 96) remain at Tailwind defaults.
