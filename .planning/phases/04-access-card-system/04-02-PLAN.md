## STRIDE Threat Register

| T-04-04 | Information Disclosure | AccessCardPrintView | mitigate | Route uses ProtectedRoute with roles=['admin','social_worker']; data fetched via authenticated API call; existing AbacGuard on backend |
| T-04-05 | Spoofing | Reprint confirmation dialog | accept | D-04 accepts simple confirmation; worker verifies identity in-person; reprint only re-displays existing data, no new auth token or escalation |
| T-04-06 | Tampering | Print view /static content | mitigate | All card data sourced from API on mount, not from URL params or localStorage — prevents data injection via URL manipulation |
| T-04-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages — all existing dependencies, skip slopcheck |
</threat_model>
