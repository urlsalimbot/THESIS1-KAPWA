<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Login form → Auth API | Email/password and OTP code cross this boundary |
| Auth API → Twilio | OTP SMS via OtpService (existing integration) |
| Coordinator browser → Dashboard API | Session-authenticated dashboard data requests |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-07 | Spoofing | POST /auth/login/otp-verify | mitigate | Temp token is JWT signed with user secret, 5-min expiry, must have otpRequired=true claim — prevents replay |
| T-06-08 | Information Disclosure | GET /dashboard for coordinator | mitigate | AbacGuard enforces barangay scoping — coordinator can only see own barangay data per assignedBarangay field |
| T-06-09 | Spoofing | SMS OTP interception | accept | SMS interception (SIM swap) is out of scope for v1; RESEARCH.md Pitfall 3 noted for future device binding |
</threat_model>
