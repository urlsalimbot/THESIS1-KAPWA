| API→Filesystem | Generated PDF/CSV/XLSX buffers in memory (not persisted to disk) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-04 | Information Disclosure | ExportController | mitigate | Each endpoint gated by @Roles() + JwtAuthGuard — unauthorized roles get 403 |
| T-06-05 | Tampering | Export format parameter | mitigate | Zod or enum validation on format param — invalid values return 400 |
| T-06-06 | Denial of Service | Large export data | accept | OOM risk for large datasets noted in RESEARCH.md Pitfall 5; mitigated by csv-stringify streaming and pdfkit buffer streaming; max rows not enforced in v1 |
